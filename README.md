# :camera: **Filter**
- - -

## :book: **Introduction**:
Program that applies filters to BMPs.
- - -

## :wrench: **Compilation**:
```
make filter
```
- - -

## :computer: **Usage**

#### **Grayscale**: 
```
./filter -g <infile.bmp> <outfile.bmp>
```
#### **Sepia**: 
```
./filter -s <infile.bmp> <outfile.bmp>
```
#### **Reflection**: 
```
./filter -r <infile.bmp> <outfile.bmp>
```
#### **Blur**: 
```
./filter -b <infile.bmp> <outfile.bmp>
```
